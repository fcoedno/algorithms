# Algorithms

Implementation of some algorithms in python

## Sorting

To sort a list of integers by using insertion sort, you may do:

    python cli.py sort  --alg insertion 7 8 6 9 2 4
    
The output shall be

    [2, 4, 6, 7, 8, 9]
    
To sort descending:

    python cli.py sort --alg insertion --desc 7 8 6 9 2 4
